Datenschutzerklärung

Diese Datenschutzerklärung gilt für die mobile App Aurum Wallet.

1. Allgemeines

Aurum Wallet ist eine lokal arbeitende App zur Verwaltung von Edelmetallbeständen. Alle Daten werden ausschließlich auf dem Gerät des Nutzers gespeichert.

2. Erhobene Daten

Die App erhebt keine personenbezogenen Daten. Es gibt keine Registrierung, kein Benutzerkonto und keine Serveranbindung.

3. Lokale Speicherung

Alle eingegebenen Daten (z. B. Käufe, Mengen, Preise) werden ausschließlich lokal auf dem Gerät gespeichert. Eine Übertragung an Dritte findet nicht statt.

4. Biometrische Daten (Face ID / Touch ID)

Die App kann Face ID bzw. Touch ID zur lokalen Entsperrung verwenden. Biometrische Daten werden nicht von der App gespeichert oder verarbeitet, sondern ausschließlich vom Betriebssystem (iOS) verwaltet.

5. Internetverbindung

Die App kann aktuelle Marktpreise für Edelmetalle abrufen. Dabei werden keine personenbezogenen Daten übertragen.
